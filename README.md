# Plan-GPT Product Page (Static Web App)

This project is a cleaned-up, deployable version of the original single-file mockup (`Produt Page plan_gpt_v3.html`), organized into separate HTML/CSS/JS files.

## Local development

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

## Build (Vercel)

```bash
npm run build
```

Vercel outputs the static site from `dist/` (configured in `vercel.json`).
